# RestoManager Changelog

## [1.0.0-alpha] - 2026-05-3

### Добавлено:
- Управление меню: добавление, удаление, редактирование блюд
- Управление заказами: создание, добавление/удаление позиций, подсчёт суммы
- Управление бронированием: бронирование, отмена, просмотр активных броней
- Финансовая отчётность: подсчёт общей выручки
- Валидация данных: проверка номеров столов, имён, времени, цен
- Модульная архитектура: разделение на models, services, utils
- Автоматические тесты (19 тестов) с покрытием всего функционала
- CI/CD: автоматический запуск тестов и линтинга при каждом push
- Документация: докстринги для всех классов и методов

### Исправлено:
- Проблема с плавающей точкой при подсчёте суммы заказа
- Ошибка доступа к приватным атрибутам в тестах
- Соответствие кода стандарту PEP 8
- И прочие мелкие недоработки, ошибки

### Технические детали:
- Python 3.10+
- Использованы аннотации типов
- Flake8 для проверки стиля кода
- autopip8 для исправления ошибок стиля кода
- GitHub Actions для CI/CD
